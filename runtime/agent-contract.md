# Agent Contract for Research Intelligence OS

## Purpose
Defines how research agents must behave to be interoperable across the swarm.

## Required Behaviors
- Follow pipeline-contract and evaluation-contract.
- Use schemas from domain repos.
- Reference human-mind models for psychology/neuro.
- Output structured JSON where possible.
- Log reproducibility metadata.
